SkinLearning

Presentación

SkinLearning es una demo que muestra una aplicación directa y escalable de las investigaciones recogidas a lo largo del TFG: Sistema de apoyo a la decisión clínica para el estudio de lesiones de piel, basado en Deep learning.

Información de contacto

    Universidad: Universidad de Sevilla
    Centro de estudio: Escuela Técnica Superior de Ingeniería Informática.
    Título del proyecto: Sistema de apoyo a la decisión cínica para el estudio de lesiones de piel basado en aprendizaje automático.
    Palabras clave: Redes neuronales convolucionales, Detección de lesiones de piel, Aprendizaje profundo, Segmentación semántica, Sistema de asistencia clínica.
    Tutor: Isabel de los Ángeles Nepomuceno Chamorro.

Herramientas

Esta aplicación web esta compuesta por:

    Un back-end basado en el framework Django, opción elegida por ser compatible con python 3.x y los sistemas creados durante la invstigación sin necesidad de usar software extra como tensorflow.js.
    Un front-end basado en bootstrap.
    Un sistema de despligue remoto: Heroku.
    Un sistema de gestión del código fuente : GitHub.


Guia de uso

La pagina de inicio de la demo consta inicialmente de un menú de navegación compuesto por tres secciones: inicio, readme del proyecto y log in. Para probar la demo completa debemos seguir el siguiente flujo:

    1. Crear un usuario. Para ello clickamos en log in, aquí y completamos el formulario.
    2. Crear un dossier con una imágen. Una vez loggeados nos aparecerá una nueva opción en la barra de navegación: "Listado de dossieres", clickamos en ella, posteriormente en el simbolo [+] y rellenamos los campos indicados: título del dossier, imágen y título de la imagen. Pinchamos en aceptar.
    3. Analizar la imágen. Clickamos de nuevo en "listado de dossieres". Nos aparecerá el nuevo dossier. Pinchamos en ver en detalles. Nos mostrará la información referente a la imágen. Damos a analizar y ...
¡Listo! nuestra lesión ya ha sido clasificada por una CNN.
